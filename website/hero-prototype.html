// THE ECOSYSTEM

Four planes.
Every one stands alone.

Task, Memory, Compute, and Tools — each a standalone product, composing only at explicit contract seams. No monorepo. No god object. No shared secret.

Release Candidate // TASK PLANE

AgentConnect

the control plane

A durable, operator-owned ledger for coding-agent work: launch, shell, audit, complete. A managed agent structurally cannot sign off on its own task.

“If it is not recorded, it did not happen.”

Release Candidate // MEMORY PLANE

BrainConnect

trusted memory ledger

Agents propose memories. Only a human promotes them to trusted. The CLI makes zero model calls — a hard determinism boundary.

“Agents can only propose, never decide.”

Shadow Mode // CONTROL MODEL

Control Model

the advisory brain

A 4B CPU-deployable model emitting strict JSON routing decisions. It advises and detects conflict; deterministic code keeps all authority.

“The model advises. Code decides.”

MVP // COMPUTE PLANE

ComputeConnect

compute control plane

Knows what compute exists, whether it's healthy, and whether a model will fit — then decides where work runs. Privacy is structural, default-deny.

“Will this fit? — answered honestly.”

MVP // TOOL PLANE

ToolConnect

tool governance

The missing layer: which tools exist, who may call them, and what happened. A fail-closed decision point with a tamper-evident, hash-chained audit.

“An error is a denial.”
